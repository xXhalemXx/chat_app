# Chat App

This is a Flutter app that allows users to chat with each other, search for other users, and modify their profiles. The app integrates Firebase for user authentication, real-time chat functionality, and cloud storage of user data and images.

## Features

- User authentication:
    - Sign up with a new account using email and password.
    - Log in with an existing account using email and password.
    - Log in with Google account.

- Profile management:
    - Update user profile information (name, phone number, and profile picture).

- Chat functionality:
    - Send and receive real-time messages to other users.
    - View a list of conversations with other users.

- User search:
    - Search for other users by name.

## Dependencies

- [fluttertoast](https://pub.dev/packages/fluttertoast): Used to display toast notifications.
- [cupertino_icons](https://pub.dev/packages/cupertino_icons): Provides Cupertino-style icons.
- [firebase_auth](https://pub.dev/packages/firebase_auth): Firebase authentication for email/password and Google Sign-In.
- [firebase_core](https://pub.dev/packages/firebase_core): Firebase core library for initialization.
- [cloud_firestore](https://pub.dev/packages/cloud_firestore): Firebase Firestore for real-time database functionality.
- [google_sign_in](https://pub.dev/packages/google_sign_in): Google Sign-In for authentication.
- [flutter_bloc](https://pub.dev/packages/flutter_bloc): State management using the BLoC pattern.
- [bloc](https://pub.dev/packages/bloc): General-purpose BLoC library.
- [get_it](https://pub.dev/packages/get_it): Dependency injection for managing singletons.
- [shared_preferences](https://pub.dev/packages/shared_preferences): Local storage for user settings.
- [flutter_native_splash](https://pub.dev/packages/flutter_native_splash): Customizable native splash screen.
- [file_picker](https://pub.dev/packages/file_picker): Pick files and images from the device.
- [firebase_storage](https://pub.dev/packages/firebase_storage): Firebase Cloud Storage for storing user images.
- [cached_network_image](https://pub.dev/packages/cached_network_image): Caches network images to improve performance.
- [auto_size_text](https://pub.dev/packages/auto_size_text): Automatically adjusts the font size to fit the available space.

## Getting Started

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/xXhalemXx/chat_app.git
   ```

2. Navigate to the project directory:

   ```bash
   cd chat_app
   ```

3. Install the Flutter dependencies:

   ```bash
   flutter pub get
   ```
5. Run the app on your emulator or device:

   ```bash
   flutter run
   ```
